# Android 应用程序屏蔽  

------

## 介绍

这是一个用于屏蔽其他应用的应用，设置你想屏蔽的应用程序列表后，当被屏蔽的程序出现在栈顶时候，会弹出一个屏蔽Activity来盖住TA。可以基于当前代码做扩展开发，如 **应用锁** 类。

原理：后台运行一个Service，定期检测当前栈顶的App是否属于被屏蔽列表，如果属于就弹出一个屏蔽Activity。
